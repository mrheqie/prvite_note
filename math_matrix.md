 * **basic variable**  
  线性方程中对应于系数矩阵主元列的变量

  * **free variabke**   
    线性方程中基本变量之外的变量

* **简化行阶梯型**  
      每一个非零的先导元素是1  
      每一先导元素1是该元素所在列的唯一非零元素  
      example:  
      |1    0    0    29|  
      |0    1   0   16|  
      |0    0   1     3|  


 * **Row reduction algorithm**  
    ```
     倍加变换  对换变换 倍乘变换

    1 . forward phase
    2 . backward phase
    ```
* **存在与唯一性定理**
   线性方程组相容的冲要条件是增广矩阵的最右列不是主元列

* **R^n**  
   含有n个元素的列向量的集合
* **Span{v}**
   通过原点的直线
* **Span{u,v}**
   通过原点的平面
