# makefile关键字

* **$^**  
    指所有的依赖文件

# gcc 编译选项
* **-g**  
    生成GDB调试信息
* **-c**  
* **-o**  
    指定编译所输出的名字



# ld 链接选项
* **-Ttext 0x80000000**  
    设置代码段起始地址

* **-o**  
    指定输出文件名字

# -objcopy 转换选项
* **-O binary**  
    指定输出为二进制文件  

* **-S**   
    不从源文件复制重定位信息和符号信息  
* **-g**  
    不从源文件复制可调试信息
